Мне пришлось поискать инструмент для загрузки изображений в рантайме и я нашёл для этих целей ассет — StandaloneFileBrowser.
Постарался оптимизировать сохранение и загрузку пинов с помощью асинхронного программирования. Сейчас всё сохраняется только при нажатии на кнопку, у загрузки есть отдельная кнопка. 
Минус в том, что каждый раз при сохранении мы перезаписываем все данные, но в будущем это можно автоматизировать, а также сделать так, 
чтобы сохранялись только добавленные пины, не перезаписывая старые — это существенно ускорит процесс сохранения и загрузки в рантайме.

Как всё работает:  

В MainScene у вас будет карта, на ней можно ставить пины по клику мышкой и просматривать их при клике по ним. 
В подробном описании будет возможность писать свой текст и нажимать кнопку "Обновить текст", чтобы обновить текстовые данные пина. 
Также есть кнопка "Изменить картинку", которая позволяет установить свою PNG-иконку.

Все данные сохраняются только при нажатии на кнопку "Сохранить".
Загружаются — соответственно — по кнопке "Загрузить".
